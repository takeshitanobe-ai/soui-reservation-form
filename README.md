# Sou-i 予約フォーム

Sou-i公式LINEのリッチメニュー用予約フォームです。

## ページ

- `index.html`
  - 右側の通常予約フォーム
- `new.html`
  - 左側の初回・新規向け予約フォーム
- `form-config.json`
  - メニュー、担当者、注意文、送信後メッセージの設定

## 編集する場所

通常の変更は `form-config.json` を編集します。

変更できる内容:

- メニュー
- 担当者
- フォーム上部の案内文
- 日時入力前の案内文
- 相談欄の見出し
- 送信後にLINEへ返す文章

## 公開後のURL

GitHub Pagesで公開すると、URLはおおむね以下の形になります。

- 通常予約フォーム: `https://GitHubユーザー名.github.io/リポジトリ名/`
- 初回予約フォーム: `https://GitHubユーザー名.github.io/リポジトリ名/new.html`

## LINE側で必要なこと

フォームを本番で使うには、LINE DevelopersのLIFF設定でエンドポイントURLを新しいURLに変更します。

- 通常予約 LIFF: `2000263820-Q6Kg85g1`
- 初回予約 LIFF: `2004257001-6krL1X3o`

リッチメニューのリンク自体は、LIFF URLのまま維持できます。

## 注意

公開直後は、必ずLINEアプリ内で送信テストをしてください。
